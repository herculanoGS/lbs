# LPS - Longest Palindromic String

Código adaptado da [versão em ANSI C](https://github.com/herculanoGS/lbs/tree/master) para rodar no ARM [Tiva™ C Series TM4C123G LaunchPad](http://www.ti.com/tool/ek-tm4c123gxl)

Você deve possuir a [IDE Energia](http://energia.nu/) instalada para executar o código.
